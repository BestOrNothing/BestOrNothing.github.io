Learning the structure of a decision network

My master dissertation is learning the structure of a decision network, so I write my understanding here about it.

Decision network

First of all, we need to know what is a decision network.
A decision network is a graphical representation of a finite sequential decision problem. Decision networks extend belief networks to include decision variables and utility. In particular, a decision network is a directed acyclic graph with chance nodes, decision nodes and a utility node.

  • Decision nodes, drawn as rectangles, represent decision variables. The agent gets to choose a value for each decision variable.
  • Chance nodes, drawn as ovals, represent random variables. These are the same as the nodes in a belief network. Each chance node has an associated domain and a conditional probability of the variable, given its parents.
  • A utility node, drawn as a diamond, represents the utility. The parents of the utility node are the variables on which the utility depends. Botch chance nodes and decision nodes can be parents of the utility node.

Note that each chance variable and each decision variable has an associated domain, but there is no domain associated with the utility node.

Adapt Bayesian networks learning algorithm to decision networks

In the last post, I introduced two Bayesian networks learning methods.
Because a decision network can be seen as a composition of a Bayesian network and a utility function, we can adapt learning methods for Bayesian networks to decision networks.
For Buntines’s method, Innes et. al (2018) introduces a method to adapt it to decision networks. They do it in two ways. Firstly, Buntine’s model assumes that the total ordering on variables is known, but in single-stage learning task the total ordering is hidden. They therefore greedily search over the space of total orderings and use integer linear programming at each step to find a most likely causal structure, which is a valid DN. Secondly, as the agent’s vocabulary of random variables expands, we need to prived new probability distributions over the larger set of possible parent sets.

References

Innes, C., Lascarides, A., Albrecht, S. V., Ramamoorthy, S., & Rosman, B. (2018). Reasoning about Unforeseen Possibilities During Policy Learning. arXiv preprint arXiv:1801.03331.
